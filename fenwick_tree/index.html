Core Idea

What bit[i] stores

bit[i] stores a suffix of prefix ending at i of length lowbit(i).

lowbit(i)

lowbit(i) = i & -i gives the least significant set bit.

Traversal jumps

Update: i += lowbit(i). Query: i -= lowbit(i).

Pseudocode Lens

Point Update

  1. i = index
  2. while i <= n:
  3. bit[i] += delta
  4. i += lowbit(i)
  5. done

Prefix Query

  1. i = index, sum = 0
  2. while i > 0:
  3. sum += bit[i]
  4. i -= lowbit(i)
  5. return sum

Range Query

  1. result = prefix(r) - prefix(l - 1)
  2. compute prefix(r)
  3. compute prefix(l - 1)
  4. return result
