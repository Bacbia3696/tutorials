Pseudocode Lens

Dijkstra(source)

  1. Initialize dist[*] = inf, dist[source] = 0; push source
  2. while priority queue is not empty:
  3. u = extract-min()
  4. if u already settled: continue; otherwise settle u
  5. for each edge (u, v, w):
  6. candidate = dist[u] + w
  7. if candidate < dist[v]:
  8. dist[v] = candidate, prev[v] = u, push/update v
  9. finish (or stop early when target settles)
